HTML 스타일 CSS사용

목록

1. CSS와 HTML의 스타일

CSS는 HTML 스타일을 더 나은 방법으로 제공하기 위해서 HTML 4부터 도입되었다.
CSS는 다음과 같은 방식으로 HTML에 첨가 될 수있다 :
  1. 인라인 - html 태그에 스타일 속성을 사용
  2. 내부 - <head> 섹션에서 <style>태그를 사용
  3. 외부 - <head> 섹션에서 <link>태그에 외부 CSS파일을 사용 : 외부CSS 파일에 CSS 구문을 사용하는 것을 권장한다.

2. 인라인 스타일

css 속성들에 대한 레퍼런스는 다음을 참조 http://www.w3schools.com/cssref/default.asp
  1. <!DOCTYPE html>
  2. <htmll>
  3. <body>
  4. <h1 style="font-family:verdana;">A heading</h1>
  5. <p style="font-family:arial;color:red;font-size:20px;">A paragraph.</p>
  6. </body>
  7. </html>

  1. <!DOCTYPE html>
  2. <htmll>
  3. <body>
  4. <h1 style="text-align:center;">Center-aligned heading</h1>
  5. <p>This is a paragraph.</p>
  6. </body>
  7. </html>

3. 내부 스타일

  1. <head>
  2. <style>
  3. body {background-color:yellow;}
  4. p {color:blue;}
  5. </style>
  6. </head>

4. 외부 스타일

  1. <head>
  2. <link rel="stylesheet" type="text/css" href="mystyle.css">
  3. </head>
mystyle.css파일
  1. body {background-color:yellow;}
  2. p {color:blue;}
※ html문서를 만들때 전달하고자 하는 content는 html태그를 문서의 스타일(레이아웃, 색깔,...등)은 css 표현하는 것을 권장
