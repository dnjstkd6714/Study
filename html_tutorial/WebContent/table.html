Html 테이블(그리드)만들기 <table>

목록

1. table

  • 테이블은 <table>태그를 사용
  • 이블 행은 <tr>을 사용
  • 행을 셀(데이터를 가지는 요소)로 분할 하는 태그는 <td>를 사용
  • 테이블의 너비, 테두리, 패딩, 셀간격, 경계설정 등은 CSS를 사용
  1. <table style="width:300px">
  2. <tr>
  3.   <td>Jill</td>
  4.   <td>Smith</td>
  5.   <td>50</td>
  6. </tr>
  7. <tr>
  8.   <td>Eve</td>
  9.   <td>Jackson</td>
  10.   <td>94</td>
  11. </tr>
  12. </table>

  1. <style>
  2. table,tr,th,td
  3. {
  4. border:1px solid black;
  5. }
  6. </style>
  8. <style>
  9. table,tr,th,td
  10. {
  11. border-collapse:collapse
  12. }
  13. </style>
  15. <style>
  16. th,td
  17. {
  18. padding:5px;
  19. }
  20. </style>
  22. <style>
  23. table
  24. {
  25. border-spacing:5px;
  26. }
  27. </style>
